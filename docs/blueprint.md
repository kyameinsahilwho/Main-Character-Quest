# **App Name**: Task Quest

## Core Features:

- Task Creation: Allow users to create tasks with titles, descriptions, due dates, and the option to add subtasks. Store the task data locally.
- Task Listing and Management: Display tasks in a clear, sortable list (e.g., by due date, priority). Enable editing, deleting, and marking tasks/subtasks as complete.
- Gamified Completion: Visually celebrate when tasks and subtasks are completed with satisfying animations (e.g., confetti) and sounds.
- Streak Tracking: Track the user's current task completion streak and longest streak. Calculate the statistics based on completion date of task . Display a celebratory message or reward when the user achieves a new longest streak.
- Completion Statistics: Calculate and display statistics such as total tasks completed (absolute and percentage), tasks completed this week/month, and estimated time saved. 

## Style Guidelines:

- Primary color: Strong violet (#9D4EDD) for an inspiring feel. 
- Background color: Very light violet (#F5EEFE). 
- Accent color: A violet-leaning blue (#577590) as a complementary contrast for callouts and stats.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clean, modern icons from a set like Material Design Icons for task categories, status indicators, and navigation.
- A split-screen layout with tasks on the left and statistics on the right.  Prioritize clarity, good information hierarchy, and generous spacing for a delightful and modern UI.
- Use subtle animations, and celebrate task completions with satisfying animations (e.g., confetti).
